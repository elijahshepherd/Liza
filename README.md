# Liza Editor

Liza Editor is an open-source graphical editor for creating and editing images and design content. It provides a user-friendly interface and a set of tools for general-purpose visual editing.

It is designed to help users easily create professional-looking assets.

Repository: https://github.com/elijahshepherd/Liza

## Features

- Intuitive graphical interface for image and design editing
- Support for common image formats
- Professional-grade editing tools
- Extensible architecture

## Installation

Installation is simple:

1. Go to the [website](https://github.com/elijahshepherd/Liza) and click the device-specific download button, **or**
2. Head to the [Releases](https://github.com/elijahshepherd/Liza/releases) page to see all available versions
3. Extract the downloaded zip file
4. Follow the platform-specific instructions:
   - **Windows**: Run the `.exe` file inside the extracted zip
   - **macOS**: Open the `.app` or `.dmg` file
   - **Linux**: Run the appropriate binary or AppImage

## Usage

Launch the application after installation. The editor provides a comprehensive set of tools for creating and editing images and design content through its graphical interface.
